Decentralised Exchange

One of the use cases of this cryptocurrency could be to decentralise the exchange of any assets,

From cars to phones and stocks as well as engineered instruments that users will be able to create if there are enough nodes interested in trading it. the Dapp would be installed as a mobile or desktop application and the users will be use it as an hybrid of ebay and trading212.

The order matching system will implement a cutting edge algorithm to ensure that enough liquidity is provided and liquidity and loiquidity proividers will be rewarded with Isokoins as they will play the roles of miners
